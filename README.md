## 🎮 About the Game

Kitchen Chaos is a fast-paced cooking game inspired by *Overcooked*. Players chop ingredients, assemble recipes, and fulfill orders against the clock. The project covers the full Unity game development pipeline — from player input and physics to UI, audio, and game state management.

---

## ✨ Features

- **Player movement** using Unity's New Input System
- **Kitchen counter interactions** — chopping, picking up, placing ingredients
- **Order & recipe system** powered by Scriptable Objects
- **Dynamic UI** — order tickets, timers, score display
- **Sound & visual feedback** for all interactions
- **Clean architecture** — event-driven design with no spaghetti dependencies
- **Scene management** — main menu, game loop, pause, and game over states

```

├── Scripts/          # All C# game logic
├── Scenes/           # Game scenes (MainMenu, GameScene)
├── Prefabs/          # Reusable game objects
├── ScriptableObjects/# Recipe and ingredient data
├── Art/              # Models, textures, materials
└── Audio/            # Sound effects and music
```
